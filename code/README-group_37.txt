There are 5 ".py" files in folder "code".
"BranchAndBound.py" and "utilForBnB.py" for Branch-and-Bound.
"Heuristics.py" for Approximation.
"LS.py" for two Local Search algorithms.

Our executable file is "exec.py".
For running this file, just type "python exec.py -inst "filename" -alg "BnB|Approx|LS1|LS2" -time "cutoff time in seconds" -seed "random seed" " in your terminal.

Attention: we use python 2.7 and the module networkx version is 1.11
